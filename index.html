Crea tu pagina web

Lo que debes ocupar es Github para hacer una pagina web. Antes de empezar a creae tu pagina web debes de conocer al menos algunas etiquetas

Etiqueta que abre y etiqueta que cierra

Antes de saber que tipos de etiquetas existen debemos saber cual es la que abre y la que cierra, La etiqueta que abre es la que se escribe asi:< > y la etiqueta que cierra se escribe de esta forma: < / >.

Tipos de etiquetas

Todo lo que sea parrafo, titulo, subtitulo, etc. Se debe escribir en la etiqueta esta etiqueta viene con la extructura, no es necesario añadirla. Otra cosa que debemos saber es que esta el p En esta eticketa puedes escribir un texto en la pagina web que estas creando. Todas las cosas que escribas van a aparecer en la pagina web, pero recuerda guardar con Ctrl+S Title Este bien añadido con la estructura del html:5. La pregunta ¿Para que es esta etiqueta? esta etiqueta es el nombre d ela pagina que seria el titulo de busqueda. Puedes cambiar el nombre de Document por otro nombre de lo que se vaya a tratar la pagina.(Recuerda presionar Ctrl + s para guardar lo que hiciste) h1 Esto es un encabezado o tambien le podemos llamar titulo. a Este se llama ancla. Para entrar a un enlace o link de aguna otra pagina o archivo de descarga, de esta forma debemos escribir en esta etiqueta a href="(link o enlace)"(Nombre del link o enlace)a, luego guardas lo que hiciste y vas a ver a la pagina web en la que te va a aparecer el nombre que le pusiste al link o enlace en color azul o celeste, para entrar al link o enlace debes presionar las letras azules o celestes, de esa forma te llevara al link de la pagina que pusiste. Si queremos ponerle color a algunas palabras debemos escribir dentro de la etiqueta head en esta eticketa al lado derecho de la etiqueta title presionamos shif + enter, de esta forma se hara un espacio para que podamos escribir lo siguiente: sty y elegimos la unica opcion que se llama style, presionamos enter y luego shif + enter para hacer un especio, despues escribimos dentro a de la etiqueta Style,ejemplo: h2 { color:(el color que queremos poner) }
