Crea tu pagina web

Lo que debes ocupar es Github para hacer una pagina web. Antes de empezar a creae tu pagina web debes de conocer al menos algunas etiquetas

Etiqueta que abre y etiqueta que cierra

Antes de saber que tipos de etiquetas existen debemos saber cual es la que abre y la que cierra, La etiqueta que abre es la que se escribe asi:<> y la etiqueta que cierra se escribe de esta forma: .

Tipos de etiquetas

Las etiquetas para hacer un titulo se debe escribir h1 tambien esta h2 y otras mas. Otra etiqueta es para escribir parrafos que va con la letra "p".
