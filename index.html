Etiqeutas para tu pagina web

Lo que debes ocupar es Github para hacer una pagina web. Antes de empezar a creae tu pagina web debes de conocer al menos algunas etiquetas

Etiqueta que abre y etiqueta que cierra

La etiqueta de la izquierda es la que abre y la de la derecha es la que cierra.

Tipos de etiquetas

Todo lo que sea parrafo, titulo, subtitulo, etc. Se debe escribir en la etiqueta body esta etiqueta viene con la extructura, no es necesario a√±adirla.

"p" En esta etiqueta puedes escribir un texto en la pagina web que estas creando. Todas las cosas que escribas van a aparecer en la pagina web, pero recuerda guardar con Ctrl+S

"h1" Esto es un encabezado o tambien le podemos llamar titulo.

"a" Este se llama ancla. Para entrar a un enlace o link de aguna otra pagina o archivo de descarga, de esta forma debemos escribir en esta etiqueta: a href="(link o enlace)"(Nombre del link o enlace)a, luego guardas lo que hiciste y vas a ver a la pagina web en la que te va a aparecer el nombre que le pusiste al link o enlace en color azul o celeste, para entrar al link o enlace debes presionar las letras azules o celestes, de esa forma te llevara al link de la pagina que pusiste.
